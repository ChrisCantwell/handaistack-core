# Contributing

Pull requests, bug reports, and practical suggestions are welcome.

## License of contributions

By submitting a pull request, patch, or other contribution, you agree that your contribution will be licensed under this repository's GPL-2.0-or-later license.

## AI-assisted contributions

AI-assisted contributions are allowed, but please disclose material AI assistance in the pull request description when practical. The goal is transparency, not purity theater.

## Practical expectations

- Keep the plugin lightweight.
- Avoid vendor lock-in and unnecessary external services.
- Do not add tracking or analytics without a clear reason and explicit documentation.
- Preserve WordPress admin usability.
- Test changes on a WordPress install before claiming they work.
