# HandAIStack Core

Core dashboard, module registry, and CTA Stack coordinator for HandAIMan WordPress tools.

Built by **TheHandAIMan with ChatGPT** as part of the broader HandAIMan / HandAIStack WordPress tooling project.

## Current stable version

`0.1.2`

## Features

- HandAIStack admin dashboard
- Module detection for related HandAIMan plugins
- Unified CTA Stack shortcode and post/podcast auto-append
- Coordinated Subscribe / Support / Contact footer panels
- Settings screen for CTA Stack behavior

## Shortcodes

- `[handaistack_cta]`
- `[handaiman_cta_stack]`
- `[ha_cta_stack]`

## Installation

1. Download the plugin ZIP from a release, or package this repository as a WordPress plugin folder.
2. Upload it in WordPress under **Plugins → Add New → Upload Plugin**.
3. Activate the plugin.
4. Configure it from the WordPress admin menu.

This plugin coordinates related HandAIMan modules. The modules remain independently installable.
